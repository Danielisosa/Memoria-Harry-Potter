# Memoria-Harry-Potter
- Es el tipico juego de memoria donde se deben voltear dos cartas y si son iguales permanecen visibles y sino deben voltearse y volver a intentar.
- Tiene un contador para el numero de intentos fallidos e intentos acertados.
- Tambien tiene un cronometro que se inicializa desde que el jugador le da a "Iniciar Juego" y se paraliza cuando el jugador ha acertado todas las 
cartas y guarda el tiempo transcurrido.
- Cuando finaliza el juego le informa  al jugador el tiempo en que realizo el juego (si lo hizo en un mayor tiempo que el juego anterior), los fallos y 
los aciertos y si vuelve a iniciar el juego se voltean todas las cartas nuevamente de manera aleatoria.

Para este juego de memoria se usaron las tecnologias Html, Css, BootsTrap, Javscript, Jquery y un poco de Json para guardar el dato del mejor tiempo jugado en el LocalStorage
